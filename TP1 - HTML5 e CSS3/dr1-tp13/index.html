Como funciona a Web? o que é uma página web?

A web tem seu termo em portugues que inicialmente era "rede mundial de computadores" ou internet. Esta é uma rede interconectada de forma descentralizada que ao ser representada em grafos se assemelha a uma "teia de aranha" o que dá origem ao termo web.

Uma pagina web é um documento de hipertexto onde é transmito informações no formato HTML e interpretado por um navegador (Chrome, Firefox, Edge) de forma compreensivel para usuários mais leigos na linguagem comum, sem códigos.

Este documento é acessado através de uma requisição de acesso via navegador através do protocolo HTTP/HTTPS, onde o cliente (usuário) faz a requisição a um outro computador, denominado servidor, que tratarpá esta requisição e fará o envio do conteúdo solicitado através de uma página no formato HTML. Assim como uma correspondencia, para chegar ao servidor correto, é necessário que o usuario digite ou clique em um link com o endereço deste computador.

Quais os papéis das linguagens HTML e CSS numa página web?

O código HTML é uma linguagem de marcação, utilizamos ela como o esqueleto do conteúdo a ser exibido para o usuario através de um navegador de internet (browser). Essa linguagem utiliza o conceito de tags com o sinal de "<" e ">", onde o conteúdo inserido entre essas tags pode apresentar funcionalidades diferentes de acordo com os padrões da linguagem.

O CSS foi concebido posteriormente ao HTML para criar estilos, características e também alguns comportamentos a esses elementos HTML, que atribuimos tags. Deste modo, eles podem ser customizadas pelo desenvolvedor de acordo com um padrão de design adotado pelo site (cores, formas, diagramação).

Explicar o que são ambientes de programação.

Nas linguagens HTML e CSS, basta ter um editor de texto (ou IDE) e um navegador como ambiente de programação, sendo o navegador para validar e verificar o comportamento do código digitado e o editor de texto ou IDE para escrever o código.
